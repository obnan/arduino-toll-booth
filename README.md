This was a project I created for my school’s robotics fair. It’s very basic, but it works flawlessly, and I learned a lot while building it.

For this project, you are going to need:

* Any Arduino or ESP32 board
* An MFRC522 sensor (any NFC sensor will work, but you will need to change the code and libraries)
* A 16x2 LiquidCrystal display with an I2C module for convenience
* A simple servo motor
* A green LED
* A red LED
* A buzzer
* A bunch of cables
